# مشروع أدوات الويب

## نظرة عامة
هذا المستودع يحتوي على مجموعة من أدوات الويب المفيدة.

## الملفات المتوفرة

### 1. معلم اللغة العربية (arabic-teacher.html)
صفحة ويب تفاعلية لتعليم اللغة العربية.

### 2. الصفحة الرئيسية (index.html)
الصفحة الرئيسية للمشروع.

### 3. أداة تحميل وسائل التواصل الاجتماعي (social-media-downloader/)
مجلد يحتوي على أداة لتحميل المحتوى من منصات التواصل الاجتماعي.

## كيفية الاستخدام

1. افتح أي من ملفات HTML في متصفح الويب الخاص بك
2. استمتع بالميزات المتاحة

## المتطلبات
- متصفح ويب حديث (Chrome, Firefox, Safari, Edge)
- اتصال بالإنترنت (إذا كانت هناك ميزات تعتمد على الخادم)

## الترخيص
جميع الحقوق محفوظة.
